### Commands

| Syntax| Description |
|--------|-------------|
| `:w` | Write (save) a file |
| `:wq` | Write (save) a file and quit |
| `i` | insert cursor into file |
| `:h` | Open help tab|
| `:sav` | Save as file as |
| `:clo` | Close current tab |
